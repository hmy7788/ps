# 배열 (Array)

**언제 씀**: 순서 있는 데이터 저장, 인덱스 접근, 슬라이싱  
**시간복잡도**: 접근 O(1), 삽입/삭제 O(N)

---

## 핵심 패턴

```python
arr = [1, 2, 3, 4, 5]
arr[0]             # 접근 O(1)
arr.append(6)      # 끝 삽입 O(1)
arr.insert(2, 10)  # 중간 삽입 O(N)
arr.pop()          # 끝 삭제 O(1)
arr.pop(0)         # 앞 삭제 O(N) ← 느림, deque 고려

# 2차원 배열
grid = [[0] * cols for _ in range(rows)]  # ← [0] * cols * rows 는 금지

# 슬라이싱
arr[1:4]      # 인덱스 1~3
arr[::-1]     # 역순
```
