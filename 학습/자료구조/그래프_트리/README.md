# 그래프 / 트리 (Graph / Tree)

**언제 씀**: 연결 관계, 최단 경로, 계층 구조, 순위 결정  
**시간복잡도**: 다익스트라 O((V+E) log V), 유니온파인드 O(α(N))

---

## 핵심 패턴

```
인접 리스트         → graph = defaultdict(list)
가중치 최단경로     → 다익스트라 (heapq)
연결 여부 확인      → 유니온 파인드
위상 정렬           → in-degree 0인 노드부터 처리 (BFS)
트리 순회           → 재귀 DFS (전위/중위/후위)
```
