# 스택 / 큐 (Stack / Queue)

**언제 씀**: 스택 — 괄호, 뒤로가기, 단조 스택 | 큐 — BFS, 순서대로 처리  
**시간복잡도**: push/pop O(1)

---

## 핵심 패턴

```
스택: list.append / list.pop()
큐:   deque.append / deque.popleft()   ← list.pop(0)은 O(n)이라 금지
단조 스택: 오른쪽에 처음으로 큰/작은 값 찾기
```
