# 해시 (Hash)

**언제 씀**: 빠른 탐색, 중복 제거, 카운팅, key-value 매핑  
**시간복잡도**: 탐색/삽입/삭제 평균 O(1)

---

## 핵심 패턴

```
중복 확인        → set
카운팅           → Counter 또는 dict
없으면 기본값    → dict.get(key, 0)
두 리스트 비교   → Counter(a) == Counter(b)
```
