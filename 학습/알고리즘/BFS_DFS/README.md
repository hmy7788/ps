# BFS / DFS

**언제 씀**: 그래프/격자 탐색, 연결 요소, 최단거리(BFS), 경우의 수(DFS)  
**시간복잡도**: O(V + E)

---

## 핵심 패턴

```
격자 맵 + 장애물 있음  → BFS (최단거리)
격자 맵 + 장애물 없음  → 맨해튼 거리
경우의 수 / 조합       → DFS + 백트래킹
연결 여부 확인         → DFS 또는 유니온 파인드
```
