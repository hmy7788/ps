# 그리디 (Greedy)

**언제 씀**: 현재 최선 선택이 전체 최선을 보장할 때  
**시간복잡도**: 보통 O(N log N) — 정렬 후 순회

---

## 핵심 패턴

```
1. 정렬 기준 잡기 (종료 시간, 크기, 빈도 등)
2. 탐욕적으로 선택
3. 반례 확인 — 항상 최적해를 보장하는지 증명 필요
```
